# Expense Tracker System (Java)

Console-based Expense Tracker application built using Java to manage and analyze daily expenses.

## Features
- Add, view, delete expenses
- Search expenses by title or category
- Sort expenses by amount and date
- Category-wise analytics and total spending
- CSV-based data persistence

## Tech Stack
- Java
- Object-Oriented Programming (OOP)
- File Handling (BufferedReader, FileWriter)
- ArrayList
- Lambda Expressions

## Project Structure
Expense.java – Model layer  
ExpenseManager.java – Business logic  
FileHandler.java – CSV persistence  
Analytics.java – Calculations  
Main.java – Controller  

## How to Run
1. Clone the repository
2. Open in any Java IDE
3. Run Main.java
